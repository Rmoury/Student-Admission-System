# Student-Admission-System
This Java-based Student Management System facilitates essential student-related operations. It consists of a Student class representing user details such as username, password, email, etc. The ValidationController class manages user validation, addition, deletion, and retrieval from a PostgreSQL database using JPA. The project incorporates servlets for handling various web requests, including user registration, login, and admin functionalities. The application demonstrates proficiency in Java EE technologies, including Servlets and JPA, following a well-structured MVC architecture for effective web development. This project provides a practical illustration of fundamental concepts in Java EE for educational purposes.
